#### Usage

修改`hostloc.py`下段代码

```python
tb = HostLoc("username", "password", "https://www.hostloc.com", "hkCM_2132_auth")
```

将`username`, `password`改成的你用户名和密码。

```python
tb = HostLoc("你的登录名", "你的密码", "https://www.hostloc.com", "hkCM_2132_auth")

```
